Foxeo

Foxeo is a Vimeo's clone and is intended for storing and viewing videos. Utilizes Ruby on Rails, React, Redux, webpack, PostgreSQL. Incorporated AWS S3 for video storage and organization.

Technology:

Rails, React, Redux, AWS, PostgreSQ, HTML Video, SCSS

Live site GitHub Repo

GroupRide

GroupRide is an event creating web app made for cyclists, made to ensure safer rides. Built using the MERN stack. We implemented a mapping system using Google's API, in order to schedule a ride. GroupRide employes React/Redux for the frontend and Express.js, BCrypt and Passport.js for authentication system.

Technology:

JavaScript, React / Redux, MongoDB, Node.js, Express.js, HTML5, CSS

Live site GitHub Repo

Crypt-Pac

Crypt-Pac is JavaScript game blending Crypt Raider and Pacman. The game uses a classical A* search algorithm to visualize a Ghost / Mummy board movement.

Technology:

JavaScript, HTML5, CSS

Live site GitHub Repo
